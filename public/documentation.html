Documentation Page

This web application will provide the user with access to info about different movies. Its Directors & genres. The user will be able to sign up and upload their personal information, along with creating a list of their favorite movies.

URL Endpoints and Request Methods

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return All Movies '/movies' GET None JSON response Object Containing all movies
Return Data about a Movie by title '/movies/:title GET None JSON object of a single movie
Return Data about a genre by name '/movies/genre/:genreName' GET None JSON res Description of genre
Return data about a Director by name '/movies/directors/:directorName' GET None JSON res w/ data about Director
Allow users to update their username '/users/:id' PUT {'Name': 'newUserName',
'movieFavorites': '[]'} 		{'Name': 'updatedUserName',
'movieFavorites': '[]'}
Allow users to add a movie to their favorites list '/users/:id/:movieTitle' POST None {'Name': 'ExampleUserName',
'movieFavorites': '[newMovie]'}
Along with text confirmation.
Allow users to remove a movie from their favorites list '/users/:id/:movieTitle' DELETE None {'Name': 'ExampleUserName',
'movieFavorites': '[newList]'}
Along with text confirmation.
Allow a user to be deleted '/users/:id' DELETE None Text confirming which user was deleted.
